About the App

HorseEventApp is a mobile app designed for riders and horse owners to manage events, profiles, and horse information in one place. 
Users can create accounts, view and edit their profiles, add horses, and access event-related features. 
The app focuses on simplicity, clean navigation, and preparing for future integration with a real backend and Azure storage.
